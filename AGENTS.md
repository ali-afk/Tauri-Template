## Project Structure

```text
src/
├── routes/
│   ├── +layout.svelte      # Root: imports CSS
│   ├── +layout.ts          # ssr=false, prerender=true — SPA mode for Tauri
│   ├── +error.svelte
│   └── (main)/
│       ├── +layout.svelte  # Wraps pages in <main>
│       └── +page.svelte    # Home page
└── lib/
    ├── components/shared/  # Reusable UI components (+ helpers/ sub-dir)
    ├── data/shared/        # DesignTokens, AppProperties, image objects
    ├── types/
    ├── scripts/
    │   ├── gen.ts              # ⚠ Core — flattens tokens → @property CSS at build time
    │   ├── media.ts            # getCurrentMedia(), queryCssProperty()
    │   ├── transition.ts        # standard() curried transition wrapper
    │   └── utils.ts            # generateId(), parseTime(), getLoadPriority(), cycleColorScale()
    ├── assets/shared/      # Images/SVGs — PascalCase barrel exports
    └── styles/
        ├── gen.css             # ⚠ Auto-generated by gen.ts — DO NOT EDIT MANUALLY
        ├── index.css           # Layer order + @imports
        ├── resets.css          # box-sizing reset; html font-size: 62.5%
        ├── variables.css       # clamp(), color-mix(), breakpoint overrides
        ├── document-loading.css
        ├── layout.css
        ├── interactive.css     # ⚠ OKLCH auto-contrast (.card, .btn)
        ├── utility.css
        └── accessibility.css
src-tauri/                  # Rust source + Tauri config
agent-memories/             # Agent memory files
docs/                       # Extended docs
```

### ⚠ Files to Treat with Extra Care

| File                             | Risk                                                                                         |
| -------------------------------- | -------------------------------------------------------------------------------------------- |
| `src/lib/scripts/gen.ts`         | Breaks all CSS vars if changed incorrectly; run `bun run src/lib/scripts/gen.ts` after changes |
| `src/lib/styles/gen.css`         | Auto-generated — never edit manually; changes will be overwritten                             |
| `src/lib/styles/interactive.css` | Breaks `.card`/`.btn` auto-contrast coloring                                                 |
| `src/lib/types/design-tokens.ts` | Structure changes break the token walker in gen.ts                                           |
